## 🎯 Productivity Hacks

### Virtual Desktops Like a Pro
1. `Win + Ctrl + D` — Create new desktop
2. `Win + Ctrl + F4` — Close current desktop
3. `Win + Ctrl + Left/Right` — Switch between desktops
4. **Pro Tip:** Right-click a desktop → Rename it (Work/Personal/Gaming)

### File Explorer Mastery
- `Ctrl + Shift + N` — Create new folder instantly
- `Alt + P` — Toggle preview pane
- `Ctrl + L` or `Alt + D` — Focus address bar
- **Quick Access:** Pin frequently used folders (right-click → Pin to Quick Access)

### Power User Commands
| Command | Action |
|---------|--------|
| `ms-settings:` | Open Settings directly from Run (`Win + R`) |
| `shell:startup` | Open Startup folder to manage auto-start apps |
| `wsreset.exe` | Reset Windows Store cache |
| `sfc /scannow` | Scan and repair system files |
| `chkdsk C: /f` | Check and fix disk errors |

### Window Management Shortcuts
- `Win + Left/Right` — Snap window left/right
- `Win + Up` — Maximize window
- `Win + Down` — Minimize/Restore window
- `Win + Shift + Left/Right` — Move window to another monitor

## 🎨 Customization Guide

### Visual Customization


Theme Settings:
  Path: Settings → Personalization → Themes
  Options:
    - Dark/Light Mode: Toggle with one click
    - Accent Color: Auto-pick from wallpaper
    - Transparency Effects: Enable for glass effect
    - Animation Effects: Disable for better performance


### Taskbar Customization
- **Auto-hide:** Right-click taskbar → Taskbar settings → Automatically hide
- **Small Taskbar:** Enable "Use smaller taskbar buttons"
- **Alignment:** Choose Center (default) or Left (classic)

### Start Menu Tweaks
1. Settings → Personalization → Start
2. Choose which folders appear (Settings, Documents, Downloads)
3. Show more/less pinned items
4. Turn off "Show recommendations for tips" to remove ads

### Lock Screen Personalization
- Settings → Personalization → Lock screen
- Choose Spotlight (bing images) or custom slideshow
- Enable "Show lock screen background picture on sign-in screen"

### Custom Cursors & Fonts
- Download cursor packs from Windows Store
- Install custom fonts (Settings → Personalization → Fonts)
- Change system font via Registry (advanced users)

### Start Menu Styling with Third-Party Tools

### WSL (Windows Subsystem for Linux)

# Install WSL 3 (2026 version)
wsl --install --version 3

# List available distros
wsl --list --online

# Install specific distro
wsl --install -d Ubuntu-24.04

# Set default version
wsl --set-default-version 3

# Access Linux files from Explorer
\\wsl$\Ubuntu\home\username

## 🛡️ Security & Privacy Essentials

### Security Checklist
- [ ] Enable **Windows Hello** (face/fingerprint unlock)
- [ ] Turn on **BitLocker** (full disk encryption)
- [ ] Enable **Controlled Folder Access** (ransomware protection)
- [ ] Turn on **Real-time Protection** (Windows Defender)
- [ ] Enable **Memory Integrity** (Core Isolation)
- [ ] Turn on **Firewall** (Public & Private networks)
- [ ] Enable **Automatic Updates** (Settings → Windows Update)
- [ ] Turn on **Smart App Control** (blocks untrusted apps)

### Privacy Settings

Recommended Privacy Configuration:
  Location:
    - Allow only essential apps (Maps, Weather, etc.)
  
  Camera & Microphone:
    - Disable for apps that don't need it
    - Enable per-app permission
  
  Activity History:
    - Disable sending to Microsoft
    - Clear history periodically
  
  Advertising ID:
    - Turn off personalized ads
  
  Diagnostics Data:
    - Choose "Required" or "Optional" based on preference
  
  Voice Activation:
    - Disable "Hey Cortana" if not using


### Network Security

# Enable Encrypted DNS (DoH)
Settings → Network & Internet → DNS → Enable "Encrypted DNS"

# Or manually via Registry
reg add HKLM\System\CurrentControlSet\Services\Dnscache\Parameters /v EnableAutoDoh /t REG_DWORD /d 1

# Flush DNS cache when having issues
ipconfig /flushdns

# Check open ports
netstat -an


### Password Management
- Use **Windows Credential Manager** to store passwords
- Enable **Microsoft Authenticator** for 2FA
- Use **Edge Password Monitor** to check for breached passwords

## ⚙️ Performance Optimization

### System Tweak Checklist

Visual Effects:
  Settings → Advanced System Settings → Performance Settings
    - Disable: Animations, Shadows, Transparency for max performance
    - Enable: Smooth fonts, Thumbnails (if needed)

Power Plan:
  - Best Performance: powercfg /setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
  - Balanced: powercfg /setactive 381b4222-f694-41f0-9685-ff5bb260df2e

Startup Programs:
  - Press Ctrl + Shift + Esc → Startup
  - Disable unnecessary apps (Spotify, Adobe, etc.)


### Storage Optimization

# Clean system files
cleanmgr /sageset:1
cleanmgr /sagerun:1

# Enable Storage Sense automatically
Set-MMAgent -StorageSenseAutoCleanup 1

# Analyze disk space
Get-PSDrive | Where-Object {$_.Used -gt 0} | 
  Select-Object Name, @{N='Used(GB)';E={[math]::Round($_.Used/1GB,2)}}

# Clear temp folders automatically via Task Scheduler


### Performance Monitoring Tools
| Tool | How to Launch | Purpose |
|------|--------------|---------|
| **Task Manager** | `Ctrl + Shift + Esc` | Processes & performance |
| **Resource Monitor** | `resmon` | Detailed resource usage |
| **Performance Monitor** | `perfmon` | Advanced system counters |
| **Game Bar** | `Win + G` | FPS, CPU, GPU, RAM stats |
| **DXDiag** | `dxdiag` | DirectX diagnostics |

### Gaming Performance
1. **Enable Game Mode:** Settings → Gaming → Game Mode → On
2. **GPU Scheduling:** Settings → System → Display → Graphics → Default graphics settings → Hardware-accelerated GPU scheduling
3. **Game DVR:** Disable background recording for more FPS
4. **Ultimate Performance Power Plan:** `powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61`

### Disk Optimization

# Defragment HDD
defrag C: /O

# Trim SSD
Optimize-Volume -DriveLetter C -ReTrim

# Check SSD health
wmic diskdrive get status,model,size,mediatype

## ❓ FAQ

### Q: Is Windows 11 free to upgrade?
**A:** Yes, if you have a genuine Windows 10 license.

### Q: How to roll back to Windows 10?
**A:** Settings → System → Recovery → Go back (10 days after upgrade).

### Q: How to disable Windows 11 ads?
**A:** Settings → Personalization → Start → Turn off "Show suggestions and ads".

### Q: How to fix slow boot times?
**A:** Disable startup apps, enable Fast Startup, and check disk health.

### Q: How to enable God Mode?
**A:** Create new folder → Rename to `GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}`.

### Q: How to use Windows without a password?
**A:** Settings → Accounts → Sign-in options → Remove password (use PIN or Windows Hello).

### Q: Can I use Android apps on Windows 11?
**A:** Yes! Amazon Appstore or Google Play alternatives (WSA).

### Q: How to get Dark Mode everywhere?
**A:** Enable Dark Mode in Settings, and use "Dark Reader" extension in Edge.
